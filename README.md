# Neil Rogers Show Transcripts

This repository contains audio transcripts of the legendary Neil Rogers Show, transcribed using OpenAI's Whisper AI model.

## About the Collection

The Neil Rogers Show was a groundbreaking talk radio program that aired in South Florida. These transcripts preserve the content of the shows for fans, researchers, and anyone interested in radio history.

## What's Inside

The repository is organized into two main folders:

- **`timestamps/`** - Transcripts with timestamps showing when each line was spoken
- **`regular/`** - Plain text transcripts without timing information

Each folder contains subfolders organized by year (e.g., `neil-rogers-show-1987`, `neil-rogers-show-1988`, etc.).

## How to Download the Files

### Option 1: Download Everything (Recommended for Beginners)

1. Click the green **Code** button near the top of this page
2. Select **Download ZIP** from the dropdown menu
3. Save the ZIP file to your computer
4. Right-click the downloaded file and select **Extract All**
5. Choose where you want to save the files

### Option 2: Download Individual Files

1. Navigate through the folders to find the file you want
2. Click on the filename to open it
3. Click the **Download** button (or right-click **Raw** and select **Save Link As**)

### Option 3: Browse Online

You can read any transcript directly on GitHub without downloading:

1. Simply click through the folders
2. Click on any `.txt` file to view it in your browser

## File Naming

Files are named using the format: `rogers-YYYYMMDD-description_base.txt`

- **YYYY** = Year
- **MM** = Month
- **DD** = Day
- **description** = Additional show information (if available)

Example: `rogers-19870414_base.txt` is the show from April 14, 1987

## Disclaimer

These transcripts were generated using AI and may contain errors. The Whisper model is highly accurate but not perfect, especially with proper names, technical terms, or unclear audio.
